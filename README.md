# Have We Met

A fun in-person scavenger hunt game where you can learn about all the different artwork
offered by the Metropolitan Mueseum of Art in New York.

This game is not officially affiliated with the Met.

## Local Development

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
